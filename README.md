# Unofficial Plex for Linux flatpak

This repo contains the Flatpak metadata and auxiliary files for an unofficial Plex for Linux build.

Note: this is not an official Plex release. This fork exists to experiment with Linux packaging changes intended to improve hardware acceleration support on Linux.
